## 1. Underminr 漏洞利用共享 CDN 架构隐藏恶意网络通信
- 分类: 漏洞与补丁 / Vulnerabilities & Patching
- 来源: SecurityWeek
- 发布时间: 2026-05-23T11:00:00+00:00
- 风险等级: 高
- 关键词: Underminr漏洞, CDN滥用, 域名前置变种, C2隐蔽通信, 网络出口策略绕过
- 命中关注词: microsoft
- 原文链接: https://www.securityweek.com/underminr-vulnerability-lets-attackers-hide-malicious-connections-behind-trusted-domains/
- 摘要来源: 大模型

- 原标题: ‘Underminr’ Vulnerability Lets Attackers Hide Malicious Connections Behind Trusted Domains

### 摘要
安全研究人员披露名为 Underminr 的新型网络攻击技术，该技术利用共享 CDN 基础设施的路由缺陷，将恶意连接伪装成受信任域名的合法流量。作为传统域名前置攻击的变种，攻击者通过构造特定的 SNI 与 Host 头，强制将请求路由至同一边缘节点的其他租户 IP，从而隐藏控制服务器通信并绕过企业网络出口策略。建议安全团队加强 DNS 解析与 TLS 握手信息的关联分析以检测此类隐蔽隧道。

### 要点
- Underminr 利用共享 CDN 边缘节点的路由机制，将恶意流量伪装为受信任域名通信。
- 该技术可有效隐藏命令控制服务器连接，并绕过基于传统域名前置缓解措施的安全策略。
- 防御需关联分析 DNS 查询、边缘 IP、SNI 与 Host 头信息，填补检测盲区。

- 攻击脚本自动下载并执行托管于 GitHub Releases 的 Linux 恶意二进制文件。
- 建议安全团队同时审查 PHP 与 JavaScript 构建工具的依赖生命周期钩子。
