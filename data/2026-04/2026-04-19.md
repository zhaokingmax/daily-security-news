## 9. 新架构 XOA 防御 Agent 间接提示注入
- 分类: AI安全与安全智能体 / AI Security & Security Agents
- 来源: 安全内参
- 发布时间: 2026-04-17
- 风险等级: 中
- 关键词: 人工智能安全, 提示注入, Agent 架构, XOA
- 命中关注词: agentic
- 原文链接: https://www.secrss.com/articles/89532
- 摘要来源: 大模型

- 原标题: XOA：一种从架构上防御Agent间接提示注入的新思路

### 摘要
弗吉尼亚理工大学提出名为 XOA 的新架构，旨在从系统结构上防御 Agent 的间接提示注入攻击。该方案主张模型不应直接读取不可信外部数据，而是生成脚本在隔离环境中处理数据。研究显示约 78.4% 的 Agent 任务可在模型不接触原始数据的情况下完成，降低了安全风险。

### 要点
- 传统 Agent 设计易受间接提示注入攻击因模型直接读取不可信数据
- XOA 架构遵循不读取原则让模型只写脚本而不接触数据
- 大部分 Agent 任务理论上无需模型直接读取原始数据即可完成
